Open Source Plane Spotting Engine

PlaneSpotted© is a tiny, open source aircraft tracking client for plane spotters and hobby coders. Try it in your browser, clone the repo, swap out APIs, and drop it into your own maps, dashboards, or side projects.

What You Get

Everything in One File

No build tools, no dependencies to install. Just HTML, CSS, and JavaScript that runs anywhere.

Real-Time Tracking

Live aircraft positions via ADS-B, updating every 5 seconds with smooth animations.

GPS Centered

Automatically centers on your location. Tap anywhere to explore other areas.

Cockpit Mode

Tap any plane to enter immersive first-person view with satellite imagery.

Weather Radar

Toggle NEXRAD precipitation overlay to see weather affecting flights.

Easy to Mod

Clean, commented code. Swap APIs, change colors, add features—it's your code now.

Mobile Ready

Touch gestures, haptic feedback, and responsive design that works on any device.

Hobby Project Disclaimer

This app uses free community APIs (airplanes.live, Iowa State weather radar) that aren't licensed for commercial use. It's meant for personal learning, tinkering, and plane spotting—not as a product or service. Please respect the data providers who make this possible.

Getting Started

Three Ways to Use It

Try the demo instantly, or grab the code and make it your own.

1

Try the Demo

Click "Try Live Demo" to run it instantly in your browser. Allow location access to see aircraft near you.

2

Clone or download from GitHub. It's a single HTML file—open it locally or host it anywhere.

3

Make It Yours

Swap out colors, add features, connect different APIs. The code is clean and well-commented.
